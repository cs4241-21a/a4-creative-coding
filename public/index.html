Info

A* is a search algorithm used to find the shortest path between between the initial and final node on a graph. It is typically used on a map to find the shortest or most cost effective way to get from Point A to Point B. This app shows a visualization of the A* search process with useful information to help understand how it works.

Usage

The panel on the right contains several options to configure the search process. Changing any of the features will affect the current search in real time, potentially making it unsolvable so careful.

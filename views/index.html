Meteorite Landings

The visualization below shows the relative mass sizes of documented meteorite landings. Using the tweakpane, the colors and the size of the circles can change. The data can also be filtered out using the type and minimum tweakpane parameters. The type parameter will choose how the meteorite was discovered (whether it was seen falling or just found on the ground). The minimum parameter will display the meteorites with that mass or greater. Note: The minimum parameter only works with all selected for type.
